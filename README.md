Running instructions:
Install XAMPP with PHP version 8.2.12 and place the project folder inside the htdocs directory.
For the full experience, add 100 audio (.wav) files to the audio-questions directory. Each audio file should be named with a number between 1 and 100 (for example, 1.wav, 2.wav, ..., 100.wav).
The numeric filename must correspond to the question number it is intended to play. For example, 25.wav will be played when Question 25 is displayed.

Application Description:
Enter any numbers between 1 and 100 in the input boxes. The ‘First Number’ is the starting number, and the ‘Last Number’ is the ending number.
Click ‘Ask Question’ to begin. To restart the process, simply change the numbers in the input boxes and click ‘Ask Question’ again, or refresh the browser page.
The application will continue asking random questions from the selected range each time you click ‘Ask Question’. As you progress, the Previous and Next buttons will appear, allowing you to navigate through the questions and review your answers.
The ‘Total Number of Questions Asked’ counter displays how many questions have been asked so far.
Each question is displayed on the screen and, if an audio (.wav) file is available, it will also be played automatically. You can replay the audio at any time to hear the question again.
Click ‘Show Answer’ to display the correct answer at the bottom of the page. To return to the top and continue asking questions, click ‘Go Back to Ask Question’.
You can record your answer by clicking ‘Start Recording’. Once you have finished answering, click ‘Stop Recording’. Then, use the Play button to listen to your recorded response and compare it with the correct answer.
Each new recording will overwrite the previous one.
You can also type your answer in the text area and compare it with the correct answer. Your typed answers are saved automatically, and you can review them later using the Previous and Next buttons.
‘You’ve answered all the questions!’ appears after all questions have been answered.
